# Weather icons for 24-hour cards

Copy SVG files from [Makin-Things/weather-icons](https://github.com/Makin-Things/weather-icons/tree/main/static) into this folder.

Cards use filenames like: `clear-day.svg`, `clear-night.svg`, `cloudy-1-day.svg`, `cloudy-1-night.svg`, `rainy-2-day.svg`, `fog-day.svg`, `wind.svg`, etc. Copy the ones you need from the repo’s `static/` directory.
